## 🔧 V25 Bug 修复

### 修复1: 深色模式切换按钮无响应

**问题原因**:
- `forEach` 在某些环境下兼容性问题
- JS函数没有正确暴露到window全局作用域

**修复方案**:
```javascript
// 改用for循环兼容所有浏览器
for (var i = 0; i < btns.length; i++) {
    btns[i].textContent = icon;
}

// 显式暴露函数到window
window.toggleDarkMode = toggleDarkMode;
```

## ✨ V26 新功能

### 1. 产品详情页「加入对比」按钮

**位置**: 在「加入购物车」按钮旁边

**功能**:
- ✅ 点击即可将当前产品加入对比列表
- ✅ 按钮有激活状态（绿色）
- ✅ 文字状态变化：「加入对比」→「已加入对比」
- ✅ 导航栏Badge数量同步更新
- ✅ 中英文双语支持

**用户体验流程**:
```
浏览产品详情
    ↓
点击「加入对比」
    ↓
按钮变绿，文字变为「已加入对比」
    ↓
导航栏对比Badge数字+1
    ↓
点击Badge进入compare.html对比页
```
