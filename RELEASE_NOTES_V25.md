# PhotonEdge V25 - 深色模式版本

## 🚀 版本概述

V25新增深色模式支持，满足光学工程师长时间查看技术参数的护眼需求。

---

## ✨ 核心新功能：深色模式 🌙

### 功能亮点

| 特性 | 说明 |
|------|------|
| 🔘 一键切换 | 导航栏月亮/太阳图标，点击即可切换 |
| 💾 记忆偏好 | localStorage保存，刷新不丢失 |
| 🧠 智能识别 | 首次访问自动跟随系统深色/浅色设置 |
| 📱 全站覆盖 | 10+页面全部支持深色模式 |
| 🎨 精心配色 | 专业护眼深色，对比度适中不刺眼 |

### 配色方案

| 元素 | 浅色模式 | 深色模式 |
|------|---------|---------|
| 主背景 | `#ffffff` | `#1a1a1a` |
| 次背景 | `#f8f9fa` | `#252525` |
| 卡片背景 | `#ffffff` | `#2a2a2a` |
| 主要文字 | `#333333` | `#e0e0e0` |
| 次要文字 | `#666666` | `#a0a0a0` |
| 边框 | `#eeeeee` | `#3a3a3a` |
| 强调色 | `#4caf50` | `#4caf50` |

### 用户体验

1. **首次访问** → 自动跟随操作系统的深色/浅色设置
2. **手动切换** → 点击导航栏月亮/太阳图标
3. **刷新页面** → 设置自动保留
4. **跨页面** → 全站统一主题
